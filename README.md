
# NetPulse - The App Version
**A lightweight, high-performance network optimizer built for gamers.**

NetPulse applies advanced, tested network tweaks to reduce latency, stabilize connections, and prioritize gaming traffic. No bloat, no subscriptions—just faster, more reliable online gameplay.

## ✨ Features
- 🔹 **Ping & Jitter Reduction** – Optimized TCP/UDP stack & routing tweaks
- 🔹 **Smart Traffic Prioritization** – QoS presets that put games first
- 🔹 **Background Process Control** – Limits non-essential network usage
- 🔹 **DNS & MTU Optimization** – Faster lookups, fewer packet drops
- 🔹 **One-Click Apply & Safe Restore** – Toggle optimizations on/off instantly
- 🔹 **Zero Bloat** – Runs silently, minimal RAM/CPU footprint

## 📥 Installation (Takes 10 Seconds)
1. Download the latest `NetPulse.exe` from the [Releases](#) page.
2. Double-click the `.exe` to run it (no installation required).
3. Grant administrator permissions when prompted.
4. Click **Optimize Now** and follow the on-screen instructions.
5. Restart your PC or network adapter if prompted. You're ready to game.

## 🎯 Quick Usage
- **Optimize:** Click the main button to apply all recommended tweaks.
- **Restore:** Click "Reset to Defaults" to revert changes instantly.
- **Presets:** Switch between `Competitive`, `Streaming`, and `Balanced` modes.
- **Monitor:** Real-time ping/jitter stats in the dashboard.

## 💻 System Requirements
- Windows 10 / 11 (64-bit)
- Administrator privileges
- Active internet connection

## ⚠️ Important Notes
- Network performance depends on your ISP, hardware, and local network setup.
- Always create a system restore point before applying system-level tweaks.
- NetPulse does not collect telemetry or personal data.
- Use responsibly. Results may vary.
